# Fussy 1.0

A git wrapper.

# Clean working directory and stage

 - ff reset

# Open and close feature branch

 - ff feature
 - ff feature complete

# Open and close hotfix branch

 - ff hotfix
 - ff hotfix complete

# Complete branch

 - ff complete

# Commit

 - ff commit

# Status

 - ff status

# Publish

 - ff publish
